## Architecture

The architecture consist of :

- `sass/`
  - `abstract/`
    - scss mixins

  - `base/`
    - reset css
    - typography
  - `components/`
    - components
    - \_button.scss
    - \_modal.scss
  - `layouts/`
    - grid
    - nav
    - footer
  - `pages/`
    - \_home.scss
    - \_checkout.scss
  - `utilities/`
    - \_flow.scss
    - sr-only.scss
